# Proyecto angular-tree-categories

Este proyecto obtiene las categorías de nuestra tienda de Prestashop y las muestra en forma de árbol a través de ángular
---

## 🚀 Instalación

- Clonar o copiar este proyecto dentro del módulo de PrestaShop (git clone https://github.com/Lucas-OrbitaDigital/angular-tree-categories.git).

- Ejecutar por consola "npm install" dentro de la carpeta "tree-categories".

---

## ⚙️ Ejecución

- Ejecutar por consola "ng serve -o" en la carpeta "tree-categories".

---

## 🛠️ Requisitos

- PHP 8.1.25

- PrestaShop 8.1.7

- Extensión mbstring habilitada.

- NodeJs 24.13.1

- AngularJs 21.1.4
